This directory contains 2 example ikc files
and one file with example timings of the various
functions that the e-puck module contains.

Epuck_inputs.ikc: This file makes the e-puck drive
	back and forth, blink and make sounds.

Epuck_camera_mic.ikc: This file attaches the Epuck
	module to the webui to show the camera and
	microphone outputs.

Timing.txt: This file shows how many milliseconds
	the various functions that can be used in
	the e-puck usually take.
